# FileOctopus MVP Usage

## Keyboard Shortcuts

| Shortcut               | Action                                     |
| ---------------------- | ------------------------------------------ |
| `Tab`                  | Switch active panel                        |
| `Enter`                | Open selected folder                       |
| `Backspace` / `Alt+Up` | Navigate to parent folder                  |
| `F5` / `Ctrl+R`        | Refresh active panel                       |
| `Delete`               | Open Trash confirmation for selected items |
| `Escape`               | Close active dialog                        |

Shortcuts are ignored while typing in text fields.

## Diagnostics

The Diagnostics panel shows app version, build profile, target OS, commit SHA when available, schema version, recovered interrupted job count, data directory, and log directory. Use the export action to create a bug-report bundle.
